# KeyCustom
3D 모델링과 AI 추천기능을 이용한 커스텀 키보드 제작 사이트

## 📌 프로젝트 소개

KeyCustom은 커스텀 키보드는 완성된 키보드를 볼수없다는 문제를 해결하기 위해 개발된 프로젝트입니다.  

커스텀 키보드는 키캡, 축, 베어본 등 여러 요소를 직접 커스터마이징하여 만든 키보드를 일컫습니다.

커스텀 요소들을 각각 따로 구매하여 완성된 키보드를 상상으로 의존하여 만들어야 한다는 것입니다.
이로 인해 완성된 제품을 받고 상상된 모습과 상이하였을 때 사용자의 만족이 떨어지거나 반품을 하는 등의 문제가 발생합니다.

이런 문제해결을 위해 3D 모델을 통해 각 요소들을 조합하여 완성된 키보드를 3D View로 보여줘 사용자의 불만족이나 반품/환불의 리크스를 개선할 수 있다고 예상됩니다.

## 🚀 주요 기능
- ✅ 주요 기능 1: 3D View 를 통한 키보드 커스텀
- ✅ 주요 기능 2: AI를 이용한 키캡, 축 , 색 조합등 추천기능
